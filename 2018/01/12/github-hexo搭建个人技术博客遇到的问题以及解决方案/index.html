github+hexo搭建个人技术博客遇到的问题以及解决方案

1.hexo删除博客中已有的文章


删除仓库里面 source/_posts/我的文章.md
执行下列命令更新博客：
hexo clean #清除缓存
hexo g == hexo generate#生成
（hexo s == hexo server#启动服务预览，其实不用执行这一步，这一步只是用来看修改后的后果，作用类似于打印预览）
hexo d == hexo deploy#部署
2.关于hexo主题的更改
我们可以去hexo的themes中去找,网址:https://hexo.io/themes/
从中选一个主题，单击主题的名字 如:next
进入到它在github上的资源，阅读README.md，安装即可。
